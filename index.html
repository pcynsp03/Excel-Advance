โจทย์

โดยกรอกแค่ รหัสสินค้า "2-113" ในช่อง G3 แล้วให้ Excel หาข้อมูลให้อัตโนมัติ

Sheet: Products (ข้อมูลต้นทาง)
B: Product CodeC: ProductD: Unit Price

// หา ชื่อสินค้า
H4 = XLOOKUP(G3, B3:B10, C3:C10)

// หา ราคา
H5 = XLOOKUP(G3, B3:B10, D3:D10)

Lookup Functions · โจทย์ที่ 3
ค้นหาข้อมูลลูกค้าจาก 3 Sheet พร้อมกัน — XLOOKUP + VSTACK
โจทย์

ข้อมูลลูกค้าถูกแยกเก็บไว้ใน 3 Sheet คือ Europe / America / Asia
ที่ Sheet Orders มีรหัสลูกค้า เช่น "VINET" และต้องการรู้ว่า VINET อยู่ประเทศอะไร

H2 = XLOOKUP(
  C2, // lookup_value = รหัสลูกค้าจาก Sheet Orders
  VSTACK(Europe!A:A, America!A:A, Asia!A:A), // lookup_array = รวม 3 Sheet เป็น 1
  VSTACK(Europe!G:G, America!G:G, Asia!G:G) // return_array = คอลัมน์ Country จาก 3 Sheet
)
1
VSTACK(…)
รวมคอลัมน์ A จากทั้ง 3 Sheet ให้เป็น 1 ชุดข้อมูลเดียวก่อน — XLOOKUP จะค้นหาในชุดนั้น
2
C2
รหัสลูกค้า เช่น "VINET" ที่อยู่ใน Sheet Orders คอลัมน์ C
3
VSTACK ที่ 2
รวมคอลัมน์ G (Country) จากทั้ง 3 Sheet — ให้ส่งค่า Country กลับมา

ทำไมถึงทำแบบนี้?
  • ถ้าค้นหาทีละ Sheet ต้องใช้ IF(IFERROR(...), IF(IFERROR(...), ...)) ซึ่งยาวมาก
  • VSTACK รวม 3 Sheet เป็น Array เดียว ให้ XLOOKUP ค้นได้ในสูตรเดียว
  • เมื่อเพิ่มข้อมูลใน Sheet ย่อย สูตรอัปเดตอัตโนมัติ

Logic Functions · โจทย์ที่ 6
ตัดเกรดนักเรียน + จัดการกรณีพิเศษ — IFS
โจทย์

มีตารางคะแนนนักเรียน ต้องการสร้างคอลัมน์ Grade โดยมีเงื่อนไขดังนี้:
<50 = F · <60 = D · <70 = C · <80 = B · ≤100 = A · ถ้าช่องว่าง = ว่าง · ถ้ากรอกผิด = "กรอกไม่ถูก"

F2 = IFS(
  E2 = "", "", // 1) ถ้าว่าง → คืนค่าว่าง
  E2 < 50, "F", // 2) น้อยกว่า 50 → F
  E2 < 60, "D", // 3) น้อยกว่า 60 → D
  E2 < 70, "C", // 4) น้อยกว่า 70 → C
  E2 < 80, "B", // 5) น้อยกว่า 80 → B
  E2 <= 100, "A", // 6) ≤100 → A
  TRUE, "กรอกไม่ถูก" // 7) อื่นๆ ทั้งหมด (เช่น 999)
)
ชื่อE: คะแนนF: เกรด (ผลลัพธ์)เหตุผล
Larry Munoz49.0F49 < 50 → F
Tristan64.8C60 ≤ 64.8 < 70 → C
Dennis100.0A100 ≤ 100 → A
Kurt Pal51.2D50 ≤ 51.2 < 60 → D
Christine(ว่าง)""เงื่อนไขแรก E2="" → คืนค่าว่าง
ข้อสังเกตสำคัญ: IFS ตรวจเงื่อนไขตามลำดับจากบนลงล่าง — ต้องเรียง <50 ก่อน <60 เสมอ ไม่เช่นนั้นคะแนน 45 จะผ่านเงื่อนไข <60 ก่อนแล้วได้ D แทน F
Summary Functions · โจทย์ที่ 7
นับสินค้าและสรุปยอดขาย — COUNTIFS & SUMIFS
โจทย์

ต้องการสร้าง Dashboard สรุปยอดขายโดยแยกตาม Category
1) นับจำนวนสินค้า Active ต่อ Category
2) รวมยอดขายตาม Category + Country + ปี พร้อมกัน

COUNTIFS — นับสินค้า Active
F2 = COUNTIFS(
  PRODUCTS!$C$2:$C$78, $E4,
  // range1=คอลัมน์ Category, crit1=ชื่อ Category จาก E4
  PRODUCTS!$J$2:$J$78, F$1
  // range2=คอลัมน์ Status, crit2=ชื่อสถานะจาก F1
)
E: CategoryF: ActiveG: Inactive
Beverages111
Condiments111
Seafood120
SUMIFS — รวมยอดขาย 3 เงื่อนไข
F2 = SUMIFS(
  ORDERDETAILS[Total], // รวมคอลัมน์นี้
  ORDERDETAILS[Category], E2,
  ORDERDETAILS[Country], $B$3,
  ORDERDETAILS[Year], $B$2
)
CategoryTotal Sales
Beverages9,548.30
Confections8,227.58
Seafood10,399.73
ความแตกต่างที่ต้องจำ: COUNTIFS → "นับ" จำนวน Cell | SUMIFS → "รวม" ตัวเลข — และ sum_range ของ SUMIFS อยู่ หน้าสุด ต่างจาก SUMIF ที่อยู่ท้าย

ค้นหารหัสลูกค้าจากชื่อ — MATCH + INDEX
โจทย์

มีตารางลูกค้า ต้องการ หารหัสลูกค้า (CustomerID) จากชื่อบริษัท "Bon app'"
โดยที่รหัสอยู่คนละคอลัมน์กับชื่อ (รหัสอยู่ A, ชื่ออยู่ B)

A: CustomerIDB: CompanyName
ALFKIAlfreds Futterkiste
ANATRAna Trujillo Emparedados
BOLIDBolido Comidas preparadas
BONAPBon app'
BOTTMBottom-Dollar Markets
วิธีคิด: ใช้ 2 ขั้นตอน
  • ใช้ MATCH หาว่า "Bon app'" อยู่แถวที่เท่าไหร่ → ได้ผลลัพธ์ 10
  • นำเลขแถว 10 ไปใส่ใน INDEX เพื่อดึงค่าจากคอลัมน์ A แถวที่ 10 → ได้ BONAP
  • เขียนรวมกันใน 1 สูตร: INDEX ครอบ MATCH ไว้ภายใน
E2 = INDEX(A:A, MATCH("Bon app'", B:B, 0) )
// MATCH หาว่า "Bon app'" อยู่แถวที่ 10 → INDEX ดึง A10 = BONAP
1
MATCH("Bon app'", B:B, 0)
ค้นหา "Bon app'" ใน Column B ด้วยการจับคู่แบบตรงเป๊ะ (0) → คืน 10 (แถวที่พบ)
2
INDEX(A:A, 10)
ดึงค่าจาก Column A แถวที่ 10 → ได้ BONAP

Dynamic Array · โจทย์ที่ 9
สร้าง Dropdown ประเทศอัตโนมัติ — SORT + DROP + UNIQUE + FILTER
โจทย์

ต้องการสร้าง Dropdown รายชื่อประเทศจากตาราง Customers
โดย ไม่ซ้ำ, เรียง A→Z, ไม่มีช่องว่าง และ อัปเดตอัตโนมัติ เมื่อเพิ่มลูกค้าใหม่

A7 = SORT(
  DROP(
    UNIQUE(
      FILTER(CUSTOMERS!G:G, CUSTOMERS!G:G <> "")
      // กรองเอาเฉพาะที่ไม่ใช่ช่องว่าง
    ),
    // หาค่าไม่ซ้ำ
  1 // ตัดแถวที่ 1 (Header) ออก
  )
)
// เรียง A→Z
1
FILTER(...<>"")
กรองเอาเฉพาะแถวที่มีข้อมูล — ตัด Cell ว่างทิ้งก่อน
2
UNIQUE(...)
ดึงเฉพาะค่าที่ไม่ซ้ำกัน — ถ้ามีลูกค้า 50 คนจาก France จะเหลือแค่ "France" ค่าเดียว
3
DROP(..., 1)
ตัดแถวแรก (Header "Country") ออก เพราะ FILTER ดึงมารวมด้วย
4
SORT(...)
เรียง A→Z อัตโนมัติ ผลลัพธ์: Argentina → Austria → Belgium → Brazil → ...
ใช้กับ Data Validation: ตั้ง Source = =$A$7# — เครื่องหมาย # หมายถึง "ทั้ง Spill Array" ดังนั้น Dropdown จะยาวขึ้น/สั้นลงอัตโนมัติตามจำนวนประเทศที่มีอยู่จริง
